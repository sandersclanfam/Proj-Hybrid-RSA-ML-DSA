
# Usage

```
./rsa_sign ./file_name
Signature created and saved to signature.bin
Note : This will run given private_key.pem is present in same directory.
```

```
./rsa_verify microsoft-edge-stable_131.0.2903.70-1_amd64.deb
Signature is valid.
Note : This will run given public_key.pem is present in same directory.
```
